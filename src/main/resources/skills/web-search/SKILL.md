# 网络搜索技能

当用户需要联网获取信息时，按以下流程执行：

## 操作流程

1. **关键词搜索**：使用 `web_search` 工具，提取用户意图中的关键词进行搜索，获取相关结果列表
2. **网页抓取**：若需深入了解某条结果，使用 `web_fetch` 工具抓取该网页的正文内容
3. **信息汇总**：综合多个来源，提炼出用户需要的答案

## 注意事项

- 搜索关键词应精炼，优先使用名词和专有词
- 对时效性问题（新闻、股价等），优先查看最新结果
- 引用信息时标注来源链接，便于用户核实
- 若搜索结果不足，尝试更换关键词或中英文双语搜索
- 抓取网页时注意区分正文与广告/导航内容
